# Quests

Quests are activity based challenges inside OceanX.

They help users discover features, build habits and earn rewards.

<Note>
Quests are not live yet. Release date TBC.
</Note>

## Quest types

| Type | Description |
| --- | --- |
| Daily quests | Short tasks refreshed each day, such as checking Scout or reviewing your portfolio |
| Trading quests | Volume or activity goals on eligible products |
| Referral quests | Milestones for inviting new users to OceanX |
| Social quests | Community participation such as joining groups or voice rooms |
| Research quests | Using the AI assistant or completing workspace analysis |
| Seasonal quests | Limited time challenges tied to active seasons or competitions |

## Rewards

Quest rewards may include:

- Points
- Badges
- Cashback boosts (meme coin trading only)
- Seasonal progress

More details will be added as quest systems expand.
