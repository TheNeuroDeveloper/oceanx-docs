# Leaderboards

Leaderboards show user rankings across OceanX activities.

<Note>
Leaderboards are not live yet. Release date TBC.
</Note>

## Leaderboard types

| Type | Description |
| --- | --- |
| Points | Overall progression ranking based on lifetime points |
| Trading volume | Rankings by total volume on eligible products |
| PnL | Rankings by profit and loss performance |
| Referrals | Rankings by referral network size and activity |
| Quest completion | Rankings by quests completed in a given period |
| Competition standings | Rankings within an active competition |
| Seasonal performance | Rankings for the current season's point totals |

## Public and opt in rankings

Some leaderboards may be public.

Some competitions may require users to opt in before being ranked.

## FAQ

### Are all leaderboards public?

Not necessarily. It depends on the competition or feature.
