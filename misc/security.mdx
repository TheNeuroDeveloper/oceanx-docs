# Security & Scam Prevention

Security is a shared responsibility.

OceanX provides non-custodial wallets, embedded wallet infrastructure and modern authentication options. Users should still protect their accounts and verify information carefully.

## Non-custodial wallets

OceanX wallets are non-custodial.

OceanX cannot access, freeze or withdraw user funds.

## Enable passkeys and MFA

Users should enable:

- Passkeys
- Multi-factor authentication
- Secure email access
- Recovery methods
- Device security

## Beware of fake tokens

The OceanX token has not launched.

There is currently no official contract address.

## Common scams

OceanX team members will never ask for:

- Your password
- Private keys
- Seed phrases
- Recovery credentials
- Remote access to your device
- Direct payments through DMs

## Official sources

Only trust official OceanX channels such as:

- Website
- Documentation
- Official X
- Official Discord
- Official Telegram

## Report suspicious activity

Report:

- Fake accounts
- Phishing links
- Impersonators
- Fake token launches
- Suspicious DMs
