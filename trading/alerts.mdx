# Alerts

Alerts help users monitor opportunities without watching the screen constantly.

OceanX plans to support flexible, criteria based alerts across wallets, tokens and markets.

## Alert channels

| Channel | Status | Description |
| --- | --- | --- |
| Push notifications | Planned | Mobile and in app alerts |
| Telegram | Planned | Bot based alert delivery |
| Discord | Planned | Server or DM notifications |
| Email | Planned | Get alert digests and event notifications by email |

## Example alert conditions

Users may create alerts for events such as:

- Wallet buys more than a set amount
- Wallet sells more than a set amount
- Market cap exceeds a threshold
- Volume spikes
- Migration occurs
- Whale enters a token
- Developer launches a token
- Watchlist token moves
- Portfolio reaches a profit target
- PnL crosses a threshold

## Scout alerts

Scout initially supports sound alerts.

Additional channels may be added over time.

## Intel alerts

Intel alerts are designed for wallet and activity monitoring.

## FAQ

### Are alerts live?

Some alert functionality may launch in phases. Scout sound alerts are planned for the initial Scout release in July 2026.

### Can I connect Telegram or Discord?

Telegram, Discord, push and email notifications are planned.
