# Trading Overview

OceanX is designed for fast, low-friction trading.

The platform combines embedded wallets, on-device signing, smart routing and optimized transaction execution.

## One-click execution

OceanX supports one-click execution.

There are no repeated wallet popups or manual confirmations for every trade.

Your embedded wallet signs on-device, then each order is:

- Simulated
- Smart-routed
- Optimized
- Submitted for execution

OceanX targets execution within one block where possible.

## OceanX routing engine

OceanX uses its own routing and execution infrastructure.

The engine is designed to improve transaction reliability, reduce friction and optimize execution paths during fast-moving market conditions.

## MEV protection

OceanX uses execution optimizations designed to reduce exposure to adverse MEV conditions, including front-running and poor execution outcomes.

## Newly migrated assets

OceanX provides optimized execution for newly migrated assets, where speed and routing quality can be especially important.

## Supported assets

OceanX supports a growing selection of digital assets.

## FAQ

### Do I need to approve every trade?

No. OceanX uses embedded wallet signing for one-click execution.

### Are there trading limits?

OceanX currently does not impose platform-level trading limits. Third-party providers or external venues may apply their own limits.

### Are fees displayed before trading?

Fees are displayed inside OceanX where applicable.
