# OceanX Docs

Mintlify-ready documentation for OceanX.

## Recommended setup

1. Create a new GitHub repository, for example `oceanx-docs`.
2. Upload these files to the repository root.
3. Sign up to Mintlify.
4. Connect the GitHub repository.
5. Set the docs domain, for example `docs.oceanx.trade`.
6. Replace placeholder changelog entries and add screenshots.

## Structure

- `mint.json` controls Mintlify navigation and theme.
- `.mdx` files contain the documentation pages.
- `images/` is reserved for screenshots, logos and diagrams.

## Notes

The OceanX token has not launched. Keep the token page visible as a scam-prevention notice or hide it until launch.
